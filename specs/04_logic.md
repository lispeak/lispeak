# Logic

In Lispeak logic is boolean.

## Values

| Word | Meaning | Additional meaning |
| ---- | ------- | ------------------ |
| `da` | true    | Yes                |
| `tu` | false   | No                 |

## Operations

| Word | Meaning |
| ---- | ------- |
| `i`  | not     |
| `ye` | and     |
| `yo` | or      |

## Examples

```
i da = tu
ye da da = da
ye da tu = tu
yo da tu = da
yo tu tu = tu
```
