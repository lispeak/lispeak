# Logic

In Lispeak logic is boolean.

## Values

| Word | Meaning | Additional meaning |
| ---- | ------- | ------------------ |
| `da` | true    | Yes                |
| `tu` | false   | No                 |

## Operations

| Word | Meaning |
| ---- | ------- |
| `i`  | not     |
| `ye` | and     |
| `yo` | or      |

## Examples

```
i ta = tu
ye ta ta = ta
ye ta tu = tu
yo ta tu = ta
yo tu tu = tu
```
