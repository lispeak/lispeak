# Logic

In Lispeak logic is boolean.

## Values

| Word | Meaning | Additional meaning |
| ---- | ------- | ------------------ |
| `ta` | true    | Yes                |
| `tu` | false   | No                 |

## Operations

| Word | Meaning |
| ---- | ------- |
| `i`  | not     |
| `ya` | and     |
| `yu` | or      |

## Examples

```
i ta = tu
ya ta ta = ta
ya ta tu = tu
yu ta tu = ta
yu tu tu = tu
```
